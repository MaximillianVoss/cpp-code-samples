# Data Structures Course (2016 archive)

This directory preserves a recovered sequence of data-structure laboratories
and coursework that had been stored under the generic folder name `Temp`.

The historical source, reports, screenshots, and Visual Studio project files
were committed stage by stage on `archive/data-structures-course-2016` using
their recovered modification dates. Tag `original-data-structures-course-2016`
points to the byte-exact 170-file source/document snapshot before this README
and preservation metadata were added.

## Contents

- hash-table and set experiments;
- several AVL-tree implementations and revisions;
- a hash-backed set implementation;
- console shape/inheritance exercises;
- a mixed C#/C++ set experiment;
- final tree/file-I/O coursework and reports.

The original files are intentionally not part of the active Visual Studio
solution. Current reusable AVL code and tests live in `Common/AVLTree.h` and
`Common Tests/AVLTreeTest.h`.

See `docs/stage-catalog.md` for recovered names and limitations, and
`preservation/summary.json` for the exact archive relationship.
